# Vue2 + Typescript + Vite Starter Template（TSX）

Starter template example for Vue2 + Typescript + Vite.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
